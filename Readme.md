Loopback Training:
1) Une premiére application basique dans laquelle on a crée un modéle "User" qui hérite de la proprorieté 
User de Loopback en suite on a fais un test avec la creation d'un utilistateur en suite se connecter avec
"access tokens".
2) Deuxiéme app basique aussi qui se compose de 2 modéles "client" et "person",
dans cette application on a traité la coté sécurité en modifiant "acls" de modéles "client".
3) Dans la 3éme App on a vu la connection entre Loopback et une base des données NOSql avec les connecteurs
BD mongoDB et on a reussi à se connecter à notre base en suite écrire et modifier dans la base. 
Aussi on a bien vu la documentation de Loopback et bien comprendre le mécanisme de creation de logique métier
et le workflow d'une application Loopback.
On a vu aussi les nouvelles fonctionnalités de Loopback 3.0 comme Loopback Studio et Arc ....
